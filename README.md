# Task Flow

A modern Project Management Application built with React, Vite, TypeScript, and Shadcn UI.

## Features

- **Authentication**: User login and registration.
- **Dashboard**: Real-time overview of projects and tasks.
- **Projects**: Create and track projects.
- **Task Board**: Kanban-style task management with Drag-and-Drop support.
- **Settings**: User profile management.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **UI**: Tailwind CSS, shadcn-ui, Lucide React
- **State Management**: React Context (Auth & Data)
- **Drag & Drop**: @hello-pangea/dnd

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  Open http://localhost:8080 to view it in the browser.

## Building for Production

```bash
npm run build
```
